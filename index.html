Corrida das Cores


: : : Início : : :


Como Funciona:

Temos quatro cores: Laranja, Amarelo, Roxo e Verde. Essas cores irão "disputar uma corrida". A cor que chegar ao final do percurso, será a ganhadora. Caso haja empate, aparecerá o botão de Reiniciar.


O Que Fazer:

Escolha em mente uma das quatro cores, e aperte play... Logo após a contagem regressiva, as cores irão "correr". Agora é só torcer pela sua!!



Especificações:

-As cores possuem um algoritmo de locomoção de forma aleatória. A cada segundo que se passa, após a contagem, elas possuem uma chance de 50% para avançar ou permanecer no mesmo local.
-Mesmo com esse algoritmo pode haver um empate entre duas cores, com uma probabilidade bem baixa.
-Caso consiga um empate com três cores, pode considerar-se com muita sorte!
-Conseguiu um empate com todas?! Caramba, quanta sorte!!!!
